# DevCamper API
> Devcamper backend application, which is a bootcamp directory website

## USAGE
Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install depedencies
```
npm install
```

## Run app
```
# Run in dev
npm run dev

# Run in production
npm start 

```
